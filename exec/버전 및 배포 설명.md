## 배포 시 특이사항

### Chrome extension 배포 관련

1. git clone 한 파일에서 npm i, npm run build 하여 build 디렉토리 생성한다.

2. 크롬 익스텐션 설치
    - 크롬 확장 프로그램 관리에서 개발자 모드로 전환한다.
    - 압축해제된 확장 프로그램을 로드한다.
    - Frontend/build 디렉토리를 선택한다.
    - 크롬 우측 상단에 SSAFY Search Engine을 고정한다.
